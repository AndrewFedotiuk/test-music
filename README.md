## Building and running on localhost:1234

First install dependencies:

```sh
yarn install
```

To run in hot module reloading mode:

```sh
yarn start
```

To create a production build:

```sh
yarn run build-prod
```

To lint code:

```sh
yarn run lint
```

To clean build:

```sh
yarn run clean
```
